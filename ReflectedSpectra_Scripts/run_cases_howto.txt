HOW TO RUN REFLECTED LIGHT SPECTRA CASES
=========================================

PREREQUISITES
-------------
1. Enable swap to extend available RAM (required — opacity database is large):
     sudo swapon /swapfile

2. Activate conda environment:
     conda activate picaso   (adjust name if different)

3. Navigate to project root (NOT into ReflectedSpectra_Scripts/):
     cd /path/to/MiniNeptuneGrid26_PostBac


RUNNING CASES
-------------
Run all 24 paper-figure cases (default batch file: batch_cases_paper_figures.json):

    for i in $(seq 0 23); do python ReflectedSpectra_Scripts/run_single_case.py $i; done

Run only the NEW cases (no prior pkl exists for these):

    for i in 3 4 5 6 7 8 12 13 14 15 16 17 21 23; do
        python ReflectedSpectra_Scripts/run_single_case.py $i
    done

Run a single case by index:

    python ReflectedSpectra_Scripts/run_single_case.py 3


CHANGING THE BATCH FILE
-----------------------
Use the --file flag with a path relative to the project root:

    python ReflectedSpectra_Scripts/run_single_case.py 14 --file ReflectedSpectra_Scripts/batch_cases_master.json

    for i in $(seq 0 48); do
        python ReflectedSpectra_Scripts/run_single_case.py $i --file ReflectedSpectra_Scripts/batch_cases_master.json
    done

Available batch files (all in ReflectedSpectra_Scripts/):
  batch_cases_paper_figures.json   <- default; 23 cases for paper contribution figures
  batch_cases_master.json          <- 49 cases; full set including H2He-rich, Archean Earth, etc.


RUNNING ON HYAK (MPI — parallel across all cases at once)
----------------------------------------------------------
Use EarthGasPlanetBatch_HyakJob.slurm for running all cases in parallel via MPI on the cluster.
Submit from the project root (NOT from inside ReflectedSpectra_Scripts/):

    sbatch ReflectedSpectra_Scripts/EarthGasPlanetBatch_HyakJob.slurm

To change which batch file is used, edit line 25 of the SLURM file:
    export BATCH_CASE_LIST="ReflectedSpectra_Scripts/batch_cases_paper_figures.json"

To run MPI locally (without SLURM, e.g. for testing):
    export BATCH_CASE_LIST="ReflectedSpectra_Scripts/batch_cases_paper_figures.json"
    mpiexec -n 4 python ReflectedSpectra_Scripts/EarthGasPlanetBatch_HyakJob.py

Note: the MPI script loads opacities on every rank — ensure swap is enabled and sufficient
RAM is available (see PREREQUISITES above).
